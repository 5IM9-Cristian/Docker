# REPOSITORIO PARA LA ENTREGA DE LA TAREA: DOCKER

En este repositorio se encuentra el ejercicio Julia con makefile anteriormente visto pero esta vez utilizando un contenedor mediante docker para asi asegurar la compatibilidad entre sistemas y equipos.

De igual manera se encuentra el ejercicio propuesto desarollado tambien utilizando docker para el contenedor.

Con esto aseguramos que el contenedor siempre va a tener las caracteristicas necesarias para que el proyecto funcione correctamente facilitando la compatibilidad y minimizando errores.
